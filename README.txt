Aby uruchomić program należy wejść w ścieżkę:
Test\LSystem\build\dist
Oraz wykonać w terminalu polecenie:
java -jar LSystem.jar

Program pisany na JRE 11.
Korzysta z JavaFX.

Program pisany jako zaliczenie projektu na studia.
Symulacja wzrostu roślin za pomocą L-systemów.

Autorzy: Kamil Kwiatkowski, Przemysław Indyka
